# Simple Interest Calculator

## Description
A simple banking application built to calculate simple interest based on user inputs. 
This project is part of the IBM Developer Skills Network Introduction to Git and GitHub course.

## Features
- Calculate simple interest from principal, rate, and time period
- Lightweight Bash script implementation
- Easy to use command-line interface

## Usage
Run the script and enter the required values when prompted:
```bash
bash simple-interest.sh
```

## Inputs
- **Principal** – The initial amount of money
- **Rate of Interest** – Annual interest rate (in %)
- **Time Period** – Duration in years

## Output
- Calculated Simple Interest value
